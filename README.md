# javaCollector
    整合java知识点，包含文章与脑图，可以用于平时复习也让更多的人体系学习java世界。

# 安装
   ## 安装依赖
    yarn add -D vuepress
   ## 构建文件
    yarn docs:build
   ## 启动服务
    yarn docs:dev

# 部署到github pages
  windows
  build.bat -> deploy.bat(需要sshkey)

# 静态页面地址
    https://leej0hn.github.io/javaCollector-online/
# 文件编辑
vscode插件markMap 画思维导图，导出html, 然后 内嵌 md。
